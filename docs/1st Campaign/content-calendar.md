## Posting Notes

- Keep feed post text short and food-focused.
- Put menu/order/reservation links in bio, story link sticker, WhatsApp status text, or post comment.
- Pair any online order CTA with the short note: verified Ghana phone required; email verification optional.
- Use mascot PNGs on clean warm backgrounds with the brand red-brown accent.
- Avoid adding too many decorative elements around the mascot.
