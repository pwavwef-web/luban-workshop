# Bao GIF Pack V2 Motion Spec

This file describes a future improved animation pass for the **Meet Bao** campaign. The v2 GIF exports are not currently checked into this repository.

Publishable files today:
- Use [gif-pack.md](gif-pack.md) for the checked-in source GIF pack.
- Use `../../assets/bao-campaign/gifs/` for the public website copies.

Recommended export approach for a future v2 pass:
- Keep the same campaign concepts as the original GIF pack.
- Add simple puppet-style motion so Bao's hands, props, and body angle change from frame to frame.
- Export MP4 or animated WebP variants later if smoother motion and smaller files are needed.
