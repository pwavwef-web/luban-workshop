Site Overview & Tech Stack

Frontend

  • Vanilla HTML / CSS / JS
  • Tailwind CSS 3.4
  • Playfair Display + Lato fonts
  • Lucide Icons

Backend

  • Firebase Cloud Functions v2
  • Cloud Firestore
  • Firebase Auth
  • Cloud Storage

Integrations

  • Arkesel SMS (OTP + notifications)
  • Nodemailer / Gmail SMTP
  • Cloudflare Turnstile
  • Google Analytics 4
  • Firebase AI (Gemini)

Multi-page static site on Firebase Hosting with serverless Cloud Functions backend. Zero frontend frameworks — pure vanilla JS.

Notification Pipeline

Email Notifications

  • New order placed (to restaurant)
  • Order cancelled (to restaurant)
  • Order confirmation (to customer)
  • Order completed (to customer)
  • New reservation (to restaurant)
  • Reservation confirmed / rejected (to customer)
  • New contact message (to restaurant)

HTML-formatted responsive templates via Nodemailer / Gmail SMTP

SMS Notifications

  • Order placed (customer + restaurant)
  • Order status updates (preparing, completed)
  • Reservation confirmed / rejected (customer)

Sent via Arkesel SMS API — Ghana +233 format normalization

All notifications triggered by Cloud Functions on Firestore document writes

Compliance

  • Privacy Policy & Terms of Use pages
